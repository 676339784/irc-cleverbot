# irc-cleverbot
An IRC bot that speaks using [cleverbot-node](https://github.com/fojas/cleverbot-node).

Install by running `npm install` and then `node bot.js`.

### Talking to the bot
Just talk to it:

    <realhuman> bot: hello
    <bot> Hello.
